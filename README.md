# Rotate Array - LeetCode Solution

## Problem Description
Rotate an array to the right by k steps.

## Solution
This repository contains an optimized solution for the rotate array problem using the reverse method.

### Algorithm
1. Reverse the entire array
2. Reverse the first k elements
3. Reverse the remaining elements

### Time Complexity
O(n)

### Space Complexity
O(1)

## How to Run
```java
javac rotatearrayoptimized.java
java rotatearrayoptimized
```

## Example
Input: [1,2,3,4,5,6,7], k = 3
Output: [5,6,7,1,2,3,4]
